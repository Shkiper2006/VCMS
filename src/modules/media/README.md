# Media module

Модуль `media` отвечает за медиатеку и reusable assets для контента.

Минимальная реализация находится в `MediaLibrary` в `src/main.ts`:

- загрузка ассета с проверкой права `media:manage`;
- список ассетов для API и админ-панели;
- поддержка изображений для блоков `image` и `gallery`.

Постоянная модель описана таблицей `media_assets`.

План развития: файловое/S3-хранилище, трансформации изображений, responsive renditions, проверка MIME-типа и связи с content blocks.
