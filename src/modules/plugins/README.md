# Plugins module

Модуль `plugins` отвечает за расширение VCMS без изменения ядра.

Минимальная реализация находится в `PluginManager` и `HookBus` в `src/main.ts`.

Плагин регистрируется через manifest и optional setup-функцию, которая подписывает обработчики на хуки CMS. Manifest может дополнительно объявлять:

- `contentTypes` — новые декларативные схемы типов контента;
- `blocks` — кастомные блоки страниц с типом `plugin:*`.

План развития: загрузка плагинов из директории `plugins/`, изоляция кода, команды включения/выключения, marketplace metadata и UI управления плагинами.
